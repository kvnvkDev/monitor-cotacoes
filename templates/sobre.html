Monitor de ações

Sobre

Esta é uma aplicação web em Python para monitorar o preço de ativos na bolsa de valores. Possui um script que roda em segundo plano capturando os dados da biblioteca yfinance e armazena alguns dados para histórico de preços. E também possui uma interface web com back-end Flask que permite ver os dados a medida que são baixados com gráficos gerados pela biblioteca matplotlib.

O arquivo initApp.py é o que inicializa a aplicação. O usuário pode agendar uma tarefa no sistema operacional para iniciar a execução desse arquivo todos os dias ou sempre que iniciar o sistema. O arquivo app.py é o script que captura os dados do yfinance e manipula o arquivo de dados. O arquivo web.py é o back-end Flask que executa as funções presentes na interface web.
No arquivo data.json é armazenado os dados extraídos de cada ticker pelo yfinance de forma que torna possível o usuário ver o histórico de preços de dias anteriores. O arquivo config.json contem as opções de configuração do usuário e os tickers que ele escolhe monitorar.

Este aplicativo usa dados extraídos do yfinance. O nome dos tickers adicionados devem seguir a nomenclatura do Yahoo Finance. Os tickers usados neste repositório são apenas como exemplo e não são uma recomendação de investimento.
